#### **Agriculture & Food Production**

* **Egg Incubator**  
  * **Link:** [https://github.com/quantenschaum/egg-incubator](https://github.com/quantenschaum/egg-incubator)  
  * **Description:** This is an Arduino-controlled egg incubator designed to maintain optimal temperature and humidity for proper egg incubation.  
* **BeeMonitor**  
  * **Link:** [https://github.com/FarmBot](https://github.com/FarmBot)  
  * **Description:** BeeMonitor is a smart beehive monitoring system that helps beekeepers track key hive parameters: temperature, humidity, weight, and sound, using sensors and open-source technologies (often Raspberry Pi and Arduino). It provides real-time information about bee colony health, helps detect environmental hazards, and contributes to making beekeeping more accessible.  
* **FarmBot**  
  * **Link:** [https://github.com/FarmBot](https://github.com/FarmBot)  
  * **Description:** This is an open-source automated farming system that allows anyone to easily grow seeds using automated tools and programs.

#### **3D Modeling & Design**

* **Blender**  
  * **Link:** [https://github.com/blender/blender](https://github.com/blender/blender)

#### **3D Printing & CNC Control**

* **PrusaSlicer**  
  * **Link:** [https://github.com/prusa3d/PrusaSlicer](https://github.com/prusa3d/PrusaSlicer)  
  * **Description:** PrusaSlicer is an open-source 3D slicer developed by Prusa Research. It converts 3D models into G-code files that 3D printers use.  
* **LinuxCNC**  
  * **Link:** [https://github.com/LinuxCNC/linuxcnc](https://github.com/LinuxCNC/linuxcnc)  
  * **Description:** LinuxCNC is a free, open-source software system for Computer-Aided Manufacturing (CAM). It allows control of CNC machines (e.g., milling or laser cutting machines).  
* **OctoPrint**  
  * **Link:** [https://github.com/OctoPrint/OctoPrint](https://github.com/OctoPrint/OctoPrint)  
  * **Description:** OctoPrint is a convenient web application for your 3D printer. It allows you to remotely control and monitor your 3D printer's operation via a web browser.

* **ROS (Robot Operating System)**  
  * **Link:** [https://github.com/ros](https://github.com/ros)  
  * **Description:** ROS is a flexible toolkit for developing robot software. It includes tools, libraries, and conventions that simplify the task of creating complex robotic systems.
